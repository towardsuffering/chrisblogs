In this post, we'll dive into some practical R programming examples using the famous iris dataset. We'll cover various techniques for data manipulation and analysis, demonstrating how R can be used to solve real-world data problems.

## 1. Median Sepal Length by Species

Let's start by calculating the median sepal length for each iris species:

```r
aggregate(Sepal.Length ~ Species, FUN=median, data=iris)
```

<OutputDisplay>Species Sepal.Length 1 setosa 5.0 2 versicolor 5.9 3 virginica 6.5</OutputDisplay>

This code uses the `aggregate()` function to group the data by species and calculate the median sepal length for each group.

## 2. Comparing Sepal and Petal Lengths

Now, let's find the average difference between sepal and petal lengths:

```r
# Approach 1: Using means
mean(iris$Sepal.Length) - mean(iris$Petal.Length)

# Approach 2: Calculate difference for each iris, then take average
iris$Length.Diff <- iris$Sepal.Length - iris$Petal.Length
mean(iris$Length.Diff)
```

<OutputDisplay>[1] 2.09 [1] 2.09</OutputDisplay>

Both approaches give the same result: on average, sepals are 2.09 cm longer than petals.

## 3. Maximum Sepal Length

Let's find the maximum sepal length and which species it belongs to:

```r
max(iris$Sepal.Length)
iris[iris$Sepal.Length == max(iris$Sepal.Length), "Species"]
```

<OutputDisplay>[1] 7.9 [1] virginica Levels: setosa versicolor virginica</OutputDisplay>

The maximum sepal length is 7.9 cm, belonging to the virginica species.

## 4. Finding a Specific Value in a Vector

This example demonstrates how to find the position of a specific value in a vector:

```r
set.seed(123, sample.kind="Rejection")
aVec <- sample(1:100)

# Using a for loop
pos44 <- NA
for (i in 1:length(aVec)) {
    if (aVec[i] == 44) {
        pos44 <- i
        break
    }
}
print(pos44)

# Using which()
which(aVec == 44)
```

<OutputDisplay>[1] 89 [1] 89</OutputDisplay>

Both methods find that 44 is at position 89 in our randomly generated vector.

## 5. Comparing Petal Length and Sepal Width

Finally, let's compare petal length and sepal width:

```r
sum(iris$Petal.Length < iris$Sepal.Width)
iris[iris$Petal.Length < iris$Sepal.Width, "Species"]
```
